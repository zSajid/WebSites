Labyrinth

Labyrinth was a project that was created within CS480 at the University of Nevada, Reno with another student known as Vance Piscetelli. The project intended goal was to utilize Physics collision, along with OpenGL methods to simulate the Labryinth game in a virtual environment.

NetCity 3D

NetCity 3D was a project that was done within CS426 (Senior Projects), the capstone class for computer science and engineering. The project was a collaboration between Zeeshan Sajid, Marco Rivas, along with Cristian Ortega. NetCity 3D was a game that was intended to teach students about the basic concepts of networking through a graphical fidelity approach as opposed to a text-book approach. It utilized the Unity game engine, Blender, along with C# as it's primary programming language"
