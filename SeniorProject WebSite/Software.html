Networking Simulator utilizes Blender, a 3D open source creation suite. For the purposes of this project, it is used to rotate, resize, manufacture different models, along with texturing them.

Networking Simulator utilizes Unity, a 3D/2D game engine for developers which focuses on two primary languages; C# and JavaScript. The game engine is focused on giving developers a good working environment to build their game upon.
